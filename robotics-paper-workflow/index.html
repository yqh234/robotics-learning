论文 1

Normalized Object Coordinate Space for Category-Level 6D Object Pose and Size Estimation（2026-06-21）

论文链接：https://arxiv.org/abs/1901.02970 · DOI：10.1109/CVPR.2019.00275

打开 PDF 论文库 查看分析结果 arXiv

category-level pose estimation6D pose estimationRGB-D posecanonical space

两篇论文介绍

类别级 6D 位姿估计的奠基论文，把未知实例的位姿与尺寸估计统一到共享 canonical coordinate space。

核心方法

  • 提出 NOCS 作为同类别物体共享的 normalized object coordinate representation。
  • 在 Mask R-CNN 框架上同时预测类别、实例 mask 与像素级 NOCS correspondence。
  • 结合 RGB-D 深度信息恢复物体的 metric 6D pose 与尺寸，并引入 mixed-reality data synthesis 扩充训练集。

我认为的机器人创新点

它把机器人感知里的一个长期难点讲清楚了：很多抓取和操作系统并不缺 pose solver，而是缺“遇到没见过的杯子、瓶子、碗，还能给出稳定位姿”的统一表示。NOCS 给了这个问题第一套真正站得住的标准答案。

局限

它仍依赖类别内共享几何先验，遇到类内形状差异很大、遮挡严重或 RGB-D 质量不足时会明显吃力；同时它更偏感知前端，不直接解决抓取质量与任务约束。

论文 2

GraspNet-1Billion: A Large-Scale Benchmark for General Object Grasping（2026-06-21）

论文链接：https://arxiv.org/abs/1912.13470 · DOI：10.1109/CVPR42600.2020.01146

打开 PDF 论文库 查看分析结果 arXiv

graspingbenchmarkgrasp detectionRGB-D clutter

两篇论文介绍

抓取方向最有代表性的公开 benchmark 之一，把真实 RGB-D clutter、密集抓取标注和统一评测正式结合起来。

核心方法

  • 构建 97,280 张真实 RGB-D 图像、190 个 clutter scene、88 个物体和超过 10 亿抓取姿态的公开 benchmark。
  • 通过物体 6D pose 标注与解析抓取计算，把 object-level grasp pose 投影到 scene coordinate，降低密集抓取标注成本。
  • 提出统一评测协议与 baseline 网络，使不同 grasp pose 预测方法能在同一标准下比较。

我认为的机器人创新点

这篇论文最值得机器人实践者重视的一点，是它把“抓取 benchmark”从玩具级闭门评测，变成了真实 RGB-D clutter 场景里的公共基础设施。后续很多 grasp detection、graspness、foundation-assisted grasping 工作，本质上都在这个底座上往前推。

局限

它主要聚焦双指抓取与数据/评测底座，本身不是任务级 grasping system；对多指灵巧手、语义条件抓取和复杂接触动力学的覆盖仍有限。

对比结论

NOCS 更偏“把物体看准”，而且是把 pose estimation 从实例级推向类别级泛化。

GraspNet-1Billion 更偏“把抓取研究放到统一公开底座上比较”，它不直接替你抓，但决定了后续很多 grasp detection 方法怎么被训练、评测和迭代。

这组一起看，能把当前论文库里“位姿感知”和“抓取 benchmark”之间缺掉的那一段补上。
